# NotesBlock
Trabalho de programação da disciplina de Programação Web 1 lecionada pelo professor José Roberto. 
O projeto consiste na criação de notas por login em um site online.
Para o projeto foi utilizado: 
- HTML 
- CSS 
- Bootstrap
- MongoDB Atlas
- IA (Gemini)
- Node.js ( Frameworks:Express; CLI: Nodemon; Bibliotecas: bcryptjs; Middlewares:para proteção de páginas e APIs, Fetch API;  EJS Embedded JavaScript)
